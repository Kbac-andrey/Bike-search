## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/bike-search-app.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will be available at `http://localhost:4200`

### Running Tests

```bash
npm test
```
